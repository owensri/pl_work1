การเปรียบเทียบการเขียนโปรแกรมแบบ OOP ใน Java, C++, และ Python

ความเหมือน

ทั้งสามภาษามีพื้นฐานการเขียนแบบ Object-Oriented Programming (OOP) ที่คล้ายคลึงกัน เช่น:

  • การสร้างคลาสและออบเจ็กต์
  • การใช้ inheritance และ polymorphism
  • การใช้ methods และ constructors

ความแตกต่าง

อย่างไรก็ตาม แต่ละภาษามีความแตกต่างในด้าน syntax และการจัดการบางอย่าง:

ลักษณะ Java C++ Python
การประกาศคลาส ใช้คีย์เวิร์ด class ใช้คีย์เวิร์ด class ใช้คีย์เวิร์ด class
การใช้ inheritance ใช้ extends ใช้ : และระบุคลาสพ่อแม่ ใช้ (parent_class)
การจัดการหน่วยความจำ Garbage Collection อัตโนมัติ จัดการด้วยตัวเอง Garbage Collection อัตโนมัติ

ตัวอย่างโค้ด

Java

class Animal { String name; Animal(String name) { this.name = name; } void speak() { System.out.println("Animal sound"); } } class Dog extends Animal { Dog(String name) { super(name); } void speak() { System.out.println("Bark"); } } public class Main { public static void main(String[] args) { Animal dog = new Dog("Buddy"); dog.speak(); // Output: Bark } }

C++

#include using namespace std; class Animal { protected: string name; public: Animal(string n) : name(n) {} virtual void speak() { cout << "Animal sound" << endl; } }; class Dog : public Animal { public: Dog(string n) : Animal(n) {} void speak() override { cout << "Bark" << endl; } }; int main() { Animal* dog = new Dog("Buddy"); dog->speak(); // Output: Bark delete dog; return 0; }

Python

class Animal: def __init__(self, name): self.name = name def speak(self): print("Animal sound") class Dog(Animal): def speak(self): print("Bark") dog = Dog("Buddy") dog.speak() # Output: Bark

สรุป

ทั้งสามภาษาแม้จะมีโครงสร้างที่คล้ายคลึงกันในแง่ของ OOP แต่ก็มีความแตกต่างกันในรายละเอียด syntax และวิธีการจัดการหน่วยความจำ รวมถึงการเขียนโค้ดบางส่วนที่ไม่เหมือนกัน
